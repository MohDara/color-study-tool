üìö Quick Reference

RGB vs CMY

RGB: Additive (screens) - Red, Green, Blue

CMY: Subtractive (printers) - Cyan, Magenta, Yellow

HSI Model

Hue: Color angle (0¬∞ = Red, 120¬∞ = Green, 240¬∞ = Blue)

Saturation: Color purity (0% = gray, 100% = pure)

Intensity: Brightness (0.299R + 0.587G + 0.144B)

Color Blindness

Protanopia: Missing red cones

Deuteranopia: Missing green cones

Tritanopia: Missing blue cones

Red-green genes on X chromosome (affects men more)
